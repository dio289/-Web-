# 公益领养 Web 平台（可继续迭代版）

这是一个可在 VS Code 中直接运行的公益领养平台前端项目，围绕“**透明流程、责任领养、公益传播**”设计。

## 本版重点（相比之前）

- 修复审核逻辑中的状态更新问题，确保“通过”后稳定标记宠物为已领养。
- 新增 **数据管理模块**：支持导出/导入 JSON 备份与一键重置演示数据。
- 首页新增审核看板指标（待审核、家访中、完成率、捐赠总额）。
- 领养中心增强可访问性（筛选控件 ARIA）与责任承诺校验。
- 继续保持移动端响应式布局，适合课程演示与社团运营试用。

## 功能模块

- 首页：公益内容卡片 + 关键指标看板
- 领养中心：筛选、搜索、紧急度优先、在线申请
- 审核中心：状态流转（待审核→家访中→通过/拒绝）
- 志愿者：报名与能力信息收集
- 捐赠公示：用途分类与公开记录
- 社区互动：话题化发布内容
- 数据管理：导入/导出/重置

## 本地运行

```bash
npm install
npm run dev
```

默认地址：`http://localhost:5173`
