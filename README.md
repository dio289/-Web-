# 公益领养 Web 平台（升级版）

本项目是一个可在 VS Code 直接运行的 **公益领养平台前端示例**，聚焦“流程透明 + 用户参与 + 公益传播”。

## 已实现模块

- 领养中心：宠物列表筛选（城市/类型/关键词）、紧急程度优先展示、在线申请。
- 审核中心：申请状态流转（待审核→家访中→通过/拒绝），通过后自动标记宠物已领养。
- 志愿者模块：在线报名、技能与经验登记。
- 捐赠公示：金额录入、用途标记（医疗/寄养/绝育/科普）、记录公开。
- 社区互动：按话题发布内容（领养日常/救助故事/科学养宠）。
- 首页内容：科普与公益传播卡片。

## 关键改进点

1. **可维护性**：将类型定义、初始数据、持久化 hook 拆分为独立文件。
2. **数据持久化**：通过 `localStorage` 保存核心业务记录。
3. **移动端适配**：卡片化布局 + 响应式网格，支持手机浏览。
4. **领养责任约束**：申请中加入“家庭同意/医疗承担/接受回访”承诺项。
5. **透明度提升**：捐赠用途结构化记录，便于后续审计展示。

## VS Code 使用方式

```bash
npm install
npm run dev
```

默认访问：`http://localhost:5173`
